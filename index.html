Guide utilisation Briefing Fdf

v1.0

La page de briefing est accessible à cette adresse :
https://grisonb.github.io/Briefing-fdf/

1. Prise en Main & Conseils

  • Pour un confort optimal, utilisez votre iPad en mode paysage.
  • Le site est optimisé pour une utilisation avec Safari, le navigateur par défaut de l'iPad.
  • Pour un accès rapide, touchez le bouton Partager dans Safari, puis "Sur l'écran d'accueil".

Gestion de la Session

Avant de commencer, comprenez cette fonction essentielle pour ne pas perdre votre travail.

  • Touchez Sauvegarder la session pour enregistrer votre progression. Les données sont sauvegardées localement dans le navigateur de votre iPad.
  • Touchez Effacer la sauvegarde pour repartir de zéro.
  • Le bouton CNCASC Login est un raccourci indispensable. Pensez à vous connecter d'abord pour pouvoir accéder sans problème aux autres pages du site CNCASC (Dispo Avions, Pélicandromes...).

2. Le Déroulé du Briefing

A. Fichiers PDF (FDS, GAAR...)

Avant de charger les PDF, assurez-vous de les avoir d'abord téléchargés depuis la boîte mail dash8secu@gmail.com dans l'application "Fichiers" de votre iPad (dans le dossier de votre choix).

Pour charger un fichier, touchez le bouton correspondant. Une fois le PDF affiché :

  • Zoomez/Dézoomez en pinçant avec deux doigts.
  • Déplacez-vous en faisant glisser avec un doigt.

Astuce iPad : Utilisez le Split View pour afficher "Fichiers" à côté de Safari et glissez-déposez le PDF sur la zone de chargement !

B. Diagramme GAAR : Analyse Automatique

La carte interactive des circuits s'affiche automatiquement lorsque vous chargez un PDF GAAR. Vous pouvez aussi l'afficher manuellement à tout moment en touchant Carte GAAR. Pour modifier un circuit :

  • Pour modifier un circuit (auto ou manuel) : vous devez d'abord toucher Ajouter un Circuit Manuel pour activer le mode édition.
  • Modifier un point : Une fois en mode édition, touchez un point sur la carte pour changer son nom ou le supprimer.

L'intelligence artificielle qui place les villes sur la carte peut parfois se tromper. Si vous constatez une erreur, signalez-la-moi en me donnant le nom exact de la ville concernée.

C. NOTAMs et Météo : L'Analyse Intelligente

Pour récupérer le fichier NOTAM, touchez l'icône d'aide ? qui décrit la procédure de connexion au site NATS pour télécharger le bulletin.

Le glisser-déposer en Split View, comme pour les autres PDF, fonctionne aussi parfaitement pour charger le fichier NOTAM.

Étape 1 : Filtre

Avant même de lire, gagnez du temps en masquant les informations non pertinentes.

  • Filtre par date : Par défaut, seuls les NOTAMs valides aujourd'hui sont affichés. Si vous préparez votre briefing la veille pour le lendemain, décochez cette case pour voir tous les NOTAMs pertinents.
  • Filtres par Mots-Clés : Par défaut, une liste de mots-clés (grues, obstacles...) est pré-sélectionnée pour masquer les NOTAMs les moins pertinents. Pour faire réapparaître une catégorie, touchez Filtres par Mots-Clés et décochez simplement la case correspondante.

Étape 2 : Analyser et Interagir

Maintenant que la liste est épurée, vous pouvez vous concentrer sur l'essentiel.

Garder la sélection Sauvegarder en PDF

Actions clés sur iPad :

  • Surligner du texte : Faites un appui long sur un mot. Ajustez les poignées bleues, puis relâchez. Le texte est automatiquement surligné.
  • Enlever un surlignage : Pour supprimer une zone surlignée, il suffit de taper une fois dessus.
  • Sélectionner un NOTAM : Touchez simplement sa case à cocher.
  • Synthétiser : Touchez Garder la sélection pour n'afficher que les éléments cochés.
  • Exporter : Touchez Sauvegarder en PDF pour générer un document propre de votre synthèse.

Tour de France : Notez que la page surveille les activités liées au Tour. Si un SUP AIP est en vigueur pour une étape, un bouton d'accès direct apparaîtra automatiquement au bas de la carte d'étape.

3. Retours & Améliorations

Pour toute remarque, demande d'amélioration ou rapport de bug, vous pouvez utiliser le dossier partagé "Retour problèmes Bfg" de l'application "Notes" (iCloud), ou me contacter directement par WhatsApp.

Pensez à bien noter votre trigramme à côté de votre remarque pour que je puisse vous recontacter si besoin de plus d'informations.
